# darkos-ogage-fn

## Introduction
This script will set your global hotkey to FN for dARKOSRE, just like AeolusUX/ArkOS-R3XS.

## Caveat 
Somehow the change doesn't seem to be persistent. Ping me on discord if you know a fix or improvise.

## Instructions
1. download as zip and extract.
2. Copy Ogage folder, install_ogage_fn.sh and restore_ogage_r3.sh into your ROMS partition/tools folder of the SD card.
3. On your R36S, under Options > Tools, you should see the files you copied.
4. Run install_ogage_fn.sh

## Other details
restore_ogage_r3.sh will restore the original R3 global hotkey.

Ogage folder contains the binaries for ogage-fn and ogage-r3. 

ogage-r3 will be redundant since there will be a backup of the original ogage binary.
